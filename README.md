# 🚀 Python QR Code Generator
A full-stack web application built with Python and Flask that generates QR codes. When scanned, the QR code opens a custom-styled "Message Unlocked" page.

## ✨ Features
- Generate QR codes from any text or URL.
- Custom-designed, mobile-responsive "View" page with CSS animations.
- Hosted live on PythonAnywhere.

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Libraries:** qrcode, pillow
- **Frontend:** HTML5, CSS3 (Google Poppins Font)
